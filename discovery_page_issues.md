# Discovery Page Design Issues

## Problems Identified

### 1. Global Timeline Component
- **Too large and prominent** - Takes up significant vertical space at top
- **Bright green background** - Visually overwhelming, competes with content
- **Large phase markers** - Circles and labels are oversized
- **Heavy visual weight** - Dominates the page hierarchy

### 2. Content Layout
- **Left sidebar with blue border** - Takes up space, creates visual imbalance
- **Bullet points in nested lists** - Creates excessive indentation
- **Large text blocks** - Questions and prompts feel overwhelming

### 3. Form Elements
- **Textarea still has multi-colored borders** - Orange dashed border visible
- **Large placeholder text** - Takes up significant space

## Recommended Fixes

1. **Simplify GlobalTimeline component** - Make it more compact and subtle
2. **Remove or minimize sidebar** - Integrate content inline
3. **Reduce text size** - Make prompts and questions more scannable
4. **Clean up borders** - Apply simplified styling consistently
5. **Reduce vertical spacing** - Make better use of viewport
